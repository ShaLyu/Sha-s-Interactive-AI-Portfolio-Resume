#Sha's Interactive AI Portfolio & Resume

这是一个基于 **Apple Design Language** 构建的交互式个人简历网页。它不仅展示了传统的职业背景，更核心的是集成了基于 **Coze** 搭建的 AI 原型项目以及基于 **Dify** 的交互式 AI 助手，旨在展示 AI 应用场景拆解与落地能力。

## 🚀 在线预览
https://gemini.google.com/share/8e40ed1c8d15

## 🛠️ 技术栈
- **前端框架**: HTML5, Tailwind CSS (基于 CDN)
- **设计风格**: Apple Human Interface Guidelines (响应式、高留白、物理交互感)
- **AI 引擎**: 
  - **Coze**: 用于构建 `break 分析助手` 与 `行研报告助手` 的 Workflow 与 Agent。
  - **Dify**: 提供网页侧边的实时交互式 AI 助手支持。

## 💡 核心模块说明

### 1. 个人核心技能栈
采用了结构化的业务闭环逻辑：
- **AI 业务场景拆解**: 识别高频痛点并转化为技术需求。
- **Workflow / Agent 设计**: 编排复杂链路，处理非线性任务。
- **上下文工程**: 针对不同模型特性的 Prompt 调优与长文本管理。
- **复杂业务分析与效果评估**: Bad Case 追踪与 ROI 量化。

### 2. AI Portfolio (作品集)
- **break 分析助手**: 针对基金运营异常 Case 的自动化排查工具。
- **行业研究报告助手**: 基于行研方法论，支持多章节并行生成与持续迭代的 Copilot。

### 3. 智能助手集成
右侧固定栏集成了由 Dify 驱动的 AI 助手，访问者可以直接通过对话了解我的项目细节与职业背景。

## 📦 如何部署
1. Fork 本仓库。
2. 进入仓库设置 `Settings > Pages`。
3. 在 `Build and deployment` 处选择 `Deploy from a branch`，分支选择 `main`，目录选择 `/ (root)`。
4. 保存后等待 1-2 分钟，即可通过生成的 URL 访问。
